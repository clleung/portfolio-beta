Amazon Web Services (AWS) QuickSight Q Walkthrough

Seattle, Washington


Timeline: June - August 2021

Team: Amazon QuickSight Documentation

Role: Technical Writing Intern

Tools: XML, Excel, Amazon QuickSight Q

Skills: Information Architecture, Documentation Writing, Database Design

Deliverables: Walkthrough Documentation, Project Artifacts, Data Model, Datasets, Example


Overview

During my summer internship at Amazon Web Services (AWS) as a Technical Writing Intern, I developed walkthrough documentation for Amazon QuickSight Q, an upcoming feature that would be released later in 2021.

The walkthrough was intended to abstract the technologies supporting the feature—natural language, deep learning, voice design—and showcase its unique selling points.

To support the walkthrough, I created an example using the feature, data sets, and a semi-realistic data model to efficiently cover the crucial steps needed to use Amazon QuickSight Q.
